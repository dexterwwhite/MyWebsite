Experience

Visa Software Engineering Intern -- Summer 2022

It is hard to put into words how much I have learned and how much I have grown as a software engineer throughout my time at Visa. My main project for the summer involved creating a Dependency Mapping Service for over 130 internal applications as well as utilizing this service within the UI of an internal website. My service consisted of a new database table, a Spring Boot service, as well as a Spring Boot RESTful API endpoint. My service replaced functions that relied on API calls that would take over a minute to compute with functions that would take under a second to compute. On our internal website (built with ReactJS), I utilized Redux in order to display the dependency and accountability information. This project had been an existing problem for the team I worked with, so my work was directly useful to the team and the company as a whole! My team used the Agile software development methodology which meant I was able to get first-hand experience with using Agile at an enterprise level. Additionally, I was taught and utilized enterprise-level git practices, which are definitely also important to know. Finally, I worked with 3 other interns throughout the summer which taught me a lot about planning and the sharing of work. Overall, I learned so much and was able to build up an invaluable amount of experience as a software engineer, and I know I will use what this internship has taught me throughout the rest of my career.
